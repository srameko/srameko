# Hi, I'm Ondřej 👋

I'm a forensic analyst and incident responder, currently working in a CERT team at a Fortune 100 company. I've been dealing with incidents ranging from misconfigurations to nation-state actors and ransomware — across on-prem, cloud, and hybrid environments.

My main focus is **log analysis**, **network forensics** (PCAP, NetFlow, Passive DNS), and **threat hunting**. I've also been a member of [Locked Shields](https://ccdcoe.org/exercises/locked-shields/) since 2014, leading a network monitoring team.

When I'm not in front of a SIEM, I run security workshops at [Czechitas](https://czechitas.cz)
